📋 Installation Steps

1

Connect Hardware

Connect your M5Stack CoreS3 to your computer via USB-C cable.

2

Enter Boot Mode (if needed)

Hold BOOT button while pressing RESET, then release BOOT. This puts the device in flash mode.

3

Click Install

Click the orange "Install" button above and select the serial port for your device.

4

Wait for Completion

The firmware will be automatically downloaded and flashed. Wait for the success message (about 1-2 minutes).

5

Start Using

Reset your device. The SeedSigner interface will appear. Create a new wallet or restore from seed!

✅ Compatible Devices

  • M5Stack CoreS3 (recommended)
  • M5Stack CoreS3 SE
  • ESP32-S3-DevKitC-1 with GC0308 camera
  • Custom ESP32-S3 boards with ILI9342C display + GC0308
