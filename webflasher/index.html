📋 Installation Steps

  1. Connect your M5Stack CoreS3 to your computer via USB-C cable
  2. Click the "Flash Firmware" button above
  3. Select the COM port when prompted (usually shows as "USB Serial" or "CP210x")
  4. Wait for the installation to complete (approximately 30-60 seconds)
  5. Restart the device - it will boot into SeedSigner automatically

✨ Features

  • BIP39 Seed Generation - Create and backup 12/24 word mnemonic seeds
  • XPUB Export - Export extended public keys (xpub/ypub/zpub) via QR
  • PSBT Signing - Sign Bitcoin transactions air-gapped
  • Address Verification - Verify receive addresses on device
  • QR Code Scanner - Built-in camera for scanning QR codes
  • Secure Element - Hardware-backed key storage
⚠️ Security Warning: This is an air-gapped signing device. Never connect it to the internet after flashing. Use QR codes for all data transfer between the device and your wallet software.
